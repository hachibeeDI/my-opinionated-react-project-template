```sh
$ yarn styleguide:server
run webpack as development
ℹ ｢wds｣: Project is running at http://localhost:6060/
...
```

で行える。

当プロジェクトではStorybookではなくStyleguidistを利用している。

詳細は公式ドキュメントで確認のこと。 https://react-styleguidist.js.org
